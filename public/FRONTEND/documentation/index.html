Preparation

First of all proceeding with Website Template installation, you need to prepare. Be sure you done all this steps listed below:

Software

We recommend you to download appropriate software to edit Website Template. Technical requirements might differ depending on template specification, therefore we list all basic stuff:

  1. First you need Adobe Photoshop (v. CS3 or higher). It is used for editing source .PSD file and is required for performing graphic design changes or adding template images.
  2. Use Sublime Text, Notepad++, Web storm (for more complex development) code editor to change source code of template files.
  3. To upload files on server you need FTP manager, like Total Commander or FileZilla.

Hosting

You can run index.html file directly from your site folder, but it won't let you explore all template functionality. You should upload template to hosting server in order to display a fully working website correctly.

You may also install Website Template to your computer locally by using localhost. In order to set up local server you may use WAMP, AppServ, MAMP or any similar software.


You cam check following tutorials on how to set up local server:

  • How to install XAMP web server
  • How to install WAMP web server

File structure

Let's talk about what's inside the package.
After template extracting you'll see the following files & folders structure:
  • Expand all
  • Collapse all
  • documentation - includes template's documentation files.
    • documentation.html - main documentation file. You are reading it at the moment.
  • site - includes all template files (.html, .css, .php...)
    • php - includes .php script for contact and subscribe form.
    • audio - includes all audio files, used in template.
    • video - includes all video files, used in template.
    • fonts - includes all custom typography and iconic fonts, used in template.
    • css - includes all .css files, used in template.
    • images - includes all images, used in .html files.
    • js - includes JavaScript libraries and jQuery plugins.
  • sources - includes template's source files.
    • psd - includes .psd Adobe Photoshop files.
    • sass - includes SASS files, used in template.
  • info.txt - includes base information.

			<div class="container">

			</div>
		</section>

</div>

<!-- Minified plugins and scripts -->
<script src="js/minified.js"></script>

</html>

JavaScript

Each template page imports 2 Javascript files:

  1. minified.js - includes JavaScript libraries and jQuery plugins. jQuery is a Javascript library that greatly reduces the amount of code that you must write. This file contain minified sources of all plugins.
  2. main.js - includes JavaScript plugins initialization and configuration.

All scripts have official documentation from developers. (all links listed in Source & Credits section)

  2. Why some videos might not work?
